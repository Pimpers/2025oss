--------------------------------------

3. main 브렌치와 dev/b 브렌치 병합
4. main 브렌치와 dev/c 브렌치 병합중 충돌 발생
5. 충돌 발생한 dev/c의 내용을 수정하고 병합 완료
6. 결과 화면 캡처와 실행 화면 캡처
7. ReadMe.md 수정
